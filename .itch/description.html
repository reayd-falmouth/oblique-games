A collection of experimental games inspired by Brian Eno’s Oblique Strategies with an aqautic theme.

Each game concept was randomly generated using an ideation tool that fused a classic game with an Oblique Strategy. The resulting prompts were processed by a Large Language Model, with text and graphics created using OpenAI’s GPT-4, GPT-3, and DALL·E 3.
