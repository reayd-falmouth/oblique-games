A compendium of experimental games inspired by Brian Eno’s Oblique Strategies, for a rapid ideation session.

The ideas were randomly generated using an ideation tool, and the text and graphics were generated using the OpenAI API text and image models GPT-4.3 and DALL·E 3.

The Philosophy

Oblique Games is inspired by Brian Eno's *Oblique Strategies*, using randomness and creative constraints to shape new gameplay experiences. Each game challenges traditional mechanics, leading to unexpected and innovative outcomes.
